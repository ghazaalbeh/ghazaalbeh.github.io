Undergraduate Project

Development of an embedded system and MATLAB-based GUI for analysis of ECG signal

 Project Report

March 2015 - June 2015

My undergraduate project was about "Development of an embedded system and MATLAB-based Graphical User Interface (GUI) for analysis of ECG signal" as a fulfillment of my B.Sc. degree supervised by Dr. Farzad Golshan.I use Single Lead Heart Rate Monitor Front End "AD8232" to sampling at a rate of 1 kHz and after digitization, fed to a microcontroller-based embedded system to convert the ECG data to a RS232 formatted serial bit-stream,Then extract the ECG signals Features like P, Q, R, S, T picks and ST segment then Implement a Matlab-Based GUI program and Plot received signal and features in Matlab.
